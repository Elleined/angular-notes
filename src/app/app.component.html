Flaticons Tutorial

  1. 1. Install the flaticon npm i @flaticon/flaticon-uicons
  2. 2. Use this link tag to every html file you create to use the icons The path relative to your file link rel="stylesheet" type="text/css" href="../../node_modules/@flaticon/flaticon-uicons/css/all/all.css"
  3. 3. See this website to get the icons Flaticon UI Icons
